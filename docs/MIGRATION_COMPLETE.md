## 🚀 Как запустить

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Настройка окружения
Создайте `.env` файл:
```env
API_TOKEN=your_bot_token
ENABLE_CACHING=true
ENABLE_MONITORING=true
```

### 3. Запуск
```bash
python main.py
```
